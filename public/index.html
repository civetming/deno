OpenEDC Server

Your OpenEDC Server is configured and running. You can already sync data that you have locally captured on openedc.org with it.

However, to be able to work with the data, please move all files from the OpenEDC App repository to the /public folder of your OpenEDC Server instance on your web server. Thereby, your OpenEDC Server will host the OpenEDC App and you can simply refresh this page to use the application.

In a future version, it is intented that the server can automatically download all required files to this directory.
